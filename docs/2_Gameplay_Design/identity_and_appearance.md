## Color & Rune Rules
| Action | Palette Effect | Rune Effect |
|--------|----------------|-------------|
| Absorb Master | +0.5 to that channel (RGB mix) | Glowing runes in **starting color** on new equip |
| Spare Master | Global desat +0.25 (→ white) | None |
| Absorb Own Color | Base cloth fully saturates + ornate runes | 'Empowered' starting style unlocks |
