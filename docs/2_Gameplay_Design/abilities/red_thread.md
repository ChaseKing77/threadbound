# Red Thread Abilities

**Primary (starting if chosen)**
- Charge Dash (air/ground burst that breaks red barriers)  
- Reverse Grapple (pull objects toward you)  
- Unravel (tear red-thread walls)

**Upgrades / Hybrid**
- Red + Blue = Flaming thread whip  
- Red + Yellow = Explosive charge jump  
- True Red (absorb Monarch only) = Ground pound that shatters entire screens
