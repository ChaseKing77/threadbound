# Core Themes & Aesthetics

### Themes
- Fate vs Free Will (the Tapestry metaphor)
- The cost of power (absorption corrupts)
- Sacrifice vs Dominion
- Community vs Isolation

### Aesthetics
Painterly 2D · Muted, fraying world contrasted by vibrant glowing threads · Silk, cloth, and loom motifs everywhere · Stained-glass storytelling sequences

Influences: Hollow Knight, Gris, Ori, Demon Slayer (cloth/flow animation), Journey (emotional minimalism)
